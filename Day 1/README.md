    @media 태그 
    {
        min 과 max 는 이상과 이하의 개념이 아니라 해상도의 개념으로 본다.
        따라서 max-width : 400px 은 "나의 최대 해상도는 400이므로 400 이하로 가면 파랑색
        
        min-width : 800px 은 "나의 최소 해상도는 800이므로 800이상으로 가면 빨간
    }
